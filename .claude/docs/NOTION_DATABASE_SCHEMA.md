# Notion database schema

Database name: `Aulas Processadas`

Suggested properties:

- `Name` — title
- `Categoria` — select
- `Tema principal` — text/select
- `Duração total` — text
- `Aulas processadas` — number
- `Status` — select: Planned, Processing, Completed, Completed with warnings, Failed
- `Tem código detectado?` — checkbox
- `Tem comandos detectados?` — checkbox
- `Pasta local` — url/text
- `Último processamento` — date
- `Modelo LLM` — text
- `Transcrição salva localmente?` — checkbox
