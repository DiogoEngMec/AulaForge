# Prompt de refatoração — AulaForge

Revise o código atual do AulaForge com foco em manutenção.

## Verificar

- funções grandes demais;
- duplicação;
- acoplamento entre módulos;
- tratamento de erro inconsistente;
- falta de testes;
- logs insuficientes;
- paths hardcoded;
- dependências externas misturadas com lógica de domínio;
- código que quebra no Windows.

## Não fazer

- Não mudar comportamento sem explicar.
- Não trocar stack sem necessidade.
- Não reescrever tudo.

## Entregável

1. diagnóstico;
2. lista de melhorias por prioridade;
3. patch incremental;
4. testes/validações;
5. riscos restantes.
