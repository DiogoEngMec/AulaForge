# Development best practices

- [ ] Leia a ordem completa de arquivos antes de implementar.
- [ ] Confirme a fase atual.
- [ ] Não implemente fora de escopo.
- [ ] Use commits pequenos.
- [ ] Rode testes.
- [ ] Não commite segredos.
- [ ] Não commite input/output de vídeos.
- [ ] Atualize documentação quando necessário.
