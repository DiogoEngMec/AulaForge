# Notion rules

- Notion integration is Phase 4, not Phase 1.
- Use Notion MCP when available.
- Search for a course page by course name before creating a new one.
- One page per course.
- Lessons should be added as Toggle Heading 1 or equivalent.
- Do not send raw transcripts to Notion.
- Do not send screenshots to Notion.
- Include OCR code/commands with timestamp and confidence warning when needed.
